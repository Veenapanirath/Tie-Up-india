# Tie-Up India

Tie-Up India is a B2B vendor marketplace platform that connects businesses with customers by enabling vendors to showcase products, manage subscriptions, and receive enquiries through a centralized dashboard.

* Features

- User and vendor authentication
- Role-based access control (Admin, Vendor, Customer)
- Product management with image uploads
- Category and brand management
- Subscription plans and payments
- Customer enquiry management
- Vendor analytics dashboard
- Email notifications
- Responsive design

## Tech Stack

## Installation

### Clone Repository

git clone <repository-url>

### Frontend Setup

cd FRONTEND
npm install
npm run dev

### Backend Setup

cd "TIEUP BACKEND"
npm install
npm run dev
